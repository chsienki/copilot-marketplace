# copilot-marketplace

A [Copilot CLI plugin marketplace](https://docs.github.com/en/copilot/how-tos/copilot-cli/customize-copilot/plugins-marketplace) for chsienki's plugins.

## Setup

```bash
copilot plugin marketplace add chsienki/copilot-marketplace
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [revue](https://github.com/chsienki/revue) | Local web-based git diff reviewer with inline comments and Copilot integration |
| [task-context](https://github.com/chsienki/task-context) | Load and manage task contexts from multiple local repos |

## Install a plugin

```bash
copilot plugin install revue@chsienki
copilot plugin install task-context@chsienki
```

## Update

```bash
copilot plugin update revue@chsienki
copilot plugin update task-context@chsienki
```
